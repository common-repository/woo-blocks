== Description ==

WooCommerce Gutenberg Block is Gutenberg block plugin what allows display products as carousel slider on posts and pages. It supports following block type : Latest Products, Featured Products, On Sale Products, Top Seller Products, Top Rated Products, Filter Tabs, Categories Tab … Each Block has fully option for layout, responsive, loading, template, custom style … you can easy to show products like you want and with Fully Ajax loading, products will load very fast.

[Live Demo](http://plugins.jmsthemes.com/wooblocks/) - [Live Document](http://wp-docs.jmsthemes.com/wooblocks/)

= 7 Block Support =
- Latest Products Block
- Featured Products Block
- On-Sale Products Block
- Top-Seller Products Block
- Top-Rated Products Block
- Filter Tab Block
- Categories Tab Block

= Advanced Setting =
- Fully Responsive
- Fully Ajax Loading
- Multi Rows & Columns
- Pagination & Navigation
- Auto-Play, Loop, rewind Carousel Slider
- Timing Setting
- 9 Loading Style Support
- More & More …

== Installation ==

1. Install the plugin from within the Dashboard or upload the directory jms-wooblocks and all its contents to the /wp-content/plugins/ directory.
2. Activate the plugin through the ‘Plugins’ menu in WordPress.

= Minimum Requirements =

* PHP 7.0 or greater is recommended
* MySQL 5.6 or greater is recommended
